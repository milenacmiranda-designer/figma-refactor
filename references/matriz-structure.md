# Estrutura Matriz

## Espaçamentos do canvas
- Entre seções principais: 240–400px
- Entre grupos: 80–160px
- Entre frames de uma sequência: 48–96px
- Entre título e conteúdo: 32–64px

## Hierarquia visual
Maior espaço = nova seção
Espaço médio = novo grupo
Espaço menor = itens relacionados

## Organização de Screens
Dentro de cada módulo:
- Linha 1: fluxo principal (esquerda para direita)
- Linha 2: estados alternativos
- Linha 3: erros
- Linha 4: empty states

## Alinhamento
- Títulos alinhados à esquerda
- Frames alinhados pelo topo
- Espaçamento uniforme
- Sem elementos diagonais sem função
